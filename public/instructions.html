How to obtain a WebGL enabled Browsers

The are multiple browser vendors now supporting webgl in the next release. Google Chrome Mozilla are currently the best options to get started. To download chrome with webgl please go to the chrome dev channel download page. Please select the dev channel link for your platform (windows,mac,linux.)

Keyboard shortcuts

  • shift+click: selects a point on the map and loads the associated map
  • ctrl+click: selects a point and returns the value of that point (in the coordinate box)
  • space bar: seperates the hemispheres, can be reversed with the reset view button
